# Agent Factory

> **A factory for automatically creating agentic workflows.**

Agent Factory is a powerful tool for generating AI agents and workflows using the Model Context Protocol (MCP). It transforms natural language descriptions into executable Python code for agentic workflows, powered by the any-agent library.

## Getting Started

Refer to the [Installation](getting-started/installation.md) for instructions on setup and usage.

## Quick Start

Get started with Agent Factory in just a few steps:

```bash
# Install Agent Factory
uv add agent-factory

# Generate your first agent
agent-factory generate --prompt "Create an agent that summarizes web content"
```

## Key Features

- **Natural Language to Code**: Transform natural language prompts directly into executable Python code
- **Smart Tooling & Integration**: Automatically identify the most relevant tools or MCP servers
- **Agent Execution**: Create runnable agents with operational instructions and dependency files
- **Agent Evaluation**: Evaluate generated agents against performance criteria
- **Multi-Turn and One-Shot Workflows**: Support for both conversational and one-shot tasks
